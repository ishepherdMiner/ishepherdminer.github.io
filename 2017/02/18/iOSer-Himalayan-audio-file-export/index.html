前言

　　平时喜欢用喜马拉雅听广播一般都离线下载后听,但是手机的空间不足,有些音频又不想删,因此想把音频文件导出,然后就可以任性删啦。 　　

环境

iPad Mini 2(8.4,已越狱)
macOS Sierra 10.12.3
喜马拉雅v5.4.57

  • cycripyt寻找喜马拉雅的进程并注入
