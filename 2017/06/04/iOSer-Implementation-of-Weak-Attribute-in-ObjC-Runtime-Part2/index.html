总结

　

弱引用查找根据 referent 属性，首次会被存储到 weak_table_t 结构体 referent 和 inline_referrers[0],当继续添加时，如果引用次数不大于 4 个保存在数组inline_referrers 中，当超过 4 个后以 hash 表的形式进行存储。移除时，根据 referrer 从 inline_referrers 中移除。
